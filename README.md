# 一登SDK使用流程

- 1.在<a href="http://superid.me" target="_blank">官网首页</a >点击【注册】完成一登开发者注册。
- 2.登录一登开发者中心，点击【SDK下载】，在一登Github上下载对应的SDK版本、Demo源码和技术文档。
- 3.登录一登开发者中心，点击【创建新应用】，填写信息。
- 4.添加应用成功后，获取APP\_ID、APP\_SECRET。
- 6.阅读技术文档并集成SDK;
- 6.测试
